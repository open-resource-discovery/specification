# Interactive Schema Explorer

The **ORD Schema Explorer** provides an interactive, force-directed graph visualization of the ORD Document and Configuration JSON Schemas. It allows you to explore the complex relationships between ORD resources, including compositions and associations.

:::tip Usage
- **Drag** nodes to reposition them.
- **Scroll** to zoom in and out.
- **Click** a node to see its full documentation and properties in the sidebar.
- **Click again** on a selected node to expand all its immediate neighbors.
- **Switch schemas** using the dropdown in the header to view either the **ORD Document** or **ORD Configuration** specification.
:::
